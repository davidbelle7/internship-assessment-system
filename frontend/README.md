# 实习生评估系统 - 前端

基于Vue.js的实习生评估系统前端项目。

## 技术栈

- Vue.js
- Vuex
- Vue Router
- Element UI
- Axios

## 项目设置

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run serve

# 构建生产版本
npm run build

# 运行测试
npm run test

# 代码检查
npm run lint
```

## 项目结构

- `src/assets`: 静态资源
- `src/components`: 通用组件
- `src/views`: 页面组件
- `src/router`: 路由配置
- `src/store`: Vuex状态管理
- `src/api`: API请求
- `src/utils`: 工具函数
